## 🖤 THE MANIFESTO

This space rejects traditional layouts in favor of an editorial experience:

- **MONOCHROME IDENTITY**
  A deep black canvas designed to focus on high-contrast structural content and eliminate visual noise.
- **TEMPORAL PRECISION**
  Consistently slow and elegant 2.4-second "switch-reveal" transitions for a calm, intentional navigation flow.
- **KINETIC INTERACTION**
  Custom-built color-inverting cursor and masked grayscale imagery for a delicate human-machine connection.
- **TYPOGRAPHIC TENSION**
  Utilizing *Syne* for brand impact and *Space Grotesk* for technical clarity and digital craftsmanship.

---

## 🛠️ ARCHITECTURAL CORE

- **Core**: React 19 + TypeScript (State-of-the-art UI patterns and type safety).
- **Localization**: `react-i18next` (Seamless bilingual integration: EN/CH).
- **Navigation**: `React Router 6` with composite animation keys.
- **Performance**: Optimized builds and asset management via Vite.
- **CI/CD**: Automated deployment via GitHub Actions.

---

## 🚀 INITIALIZE ENVIRONMENT

To replicate or explore this architecture locally:

```bash
# Clone the repository
git clone https://github.com/chichi0418/chiwebsite.git

# Install dependencies
npm install

# Execute development server
npm run dev

# Generate production archive
npm run build
```
