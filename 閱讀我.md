## 🖤 設計宣言 (THE MANIFESTO)

本計畫背離傳統的排版框架，轉而追求具備雜誌感（Editorial）的感官體驗：
- **單色視覺認同**：深邃的純黑畫布，旨在強化高對比內容的結構與張力。
- **時間精準度**：2.4 秒的「緩慢顯現」轉場動畫，創造沉穩、刻意的導航流動。
- **動態互動**：自定義的反相鼠標圓點與帶有遮罩的黑白影像處理。
- **字體張力**：採用 *Syne* 展現品牌力，並以 *Space Grotesk* 傳遞技術的洗練感。

## 🛠️ 技術核心 (ARCHITECTURAL CORE)

- **核心框架**：React 19 + TypeScript (採用現代化 UI 設計模式)。
- **國際化**：`react-i18next` (雙語系 EN/CH 深度整合)。
- **導航邏輯**：`React Router 6` 結合複合式動畫 Key 觸發機制。
- **動態效果**：基於 Intersection Observer 的自定義 CSS 貝茲曲線動畫。
- **自動化部署**：透過 GitHub Actions 實現 CI/CD 流。

## 🚀 初始化開發環境

若要在本地環境複製或探索此架構：

```bash
# 複製儲存庫
git clone https://github.com/chichi0418/chiwebsite.git

# 安裝依賴套件
npm install

# 啟動開發伺服器
npm run dev

# 建立生產環境封裝
npm run build
```
