Overview of HTML and CSS Elements Used

HTML Elements

  • <div> - Defines a division or section in an HTML document
  • <h1>, <h2> - Represents headings (main and subheading)
  • <p> - Defines a paragraph
  • <a> - Defines a hyperlink
  • <img> - Embeds an image
  • <button> - Defines a clickable button
  • <footer> - Represents the footer section of a page
  • <link> - Defines a link to an external resource (e.g., CSS)
  • <meta> - Provides metadata about the HTML document (charset, viewport)
  • <title> - Sets the title of the document, displayed in the browser tab
  • <header> - Defines the header section of the document
  • <ul>, <li> - Creates an unordered list and list items
  • <style> - Embeds internal CSS styles within the HTML document
  • <target="_blank"> - Opens links in a new tab or window

CSS Styles Used

  • font-family - Specifies the font for text (e.g., Arial)
  • margin - Sets the outer spacing around elements (e.g., 0, 5px, 10px)
  • line-height - Sets the space between lines of text (e.g., 1.6)
  • color - Defines the color of the text (e.g., #333, white, black, #3498db)
  • justify-content - Aligns items horizontally (e.g., center)
  • align-items - Aligns items vertically (e.g., center)
  • background-color - Sets the background color (e.g., rgba(128, 128, 128, 0.15), #333)
  • position - Specifies how an element is positioned (e.g., absolute, fixed)
  • transition - Adds transition effects (e.g., 0.3s ease)
  • border-radius - Rounds the corners of elements (e.g., 50%, 10%)
  • font-size - Defines the size of the font (e.g., 16px, 48px, 20px)
  • max-width - Limits the width of elements (e.g., 600px)
  • bottom - Specifies the bottom position (e.g., `bottom: 0`)
  • cursor - Defines the type of cursor when hovering (e.g., pointer)
  • hover effects - Changes the element’s properties when hovered (e.g., opacity, color, underline)
  • display - Defines how an element is displayed (e.g., flex)
  • flex-direction - Specifies the direction of the flex items (e.g., column)
  • gap - Specifies space between flex items (e.g., 20px)
  • box-shadow - Adds shadow to elements (e.g., 0 -2px 5px rgba(0,0,0,0.1))
  • @media - Defines media queries for responsive design (e.g., max-width: 768px)
  • @keyframes - Defines the steps for a CSS animation (e.g., background color transition)
  • animation - Applies an animation (e.g., backgroundChange 10s infinite)
  • list-style-type - Specifies the bullet style of lists (e.g., none)
  • text-decoration - Controls the decoration of text (e.g., none, underline)
